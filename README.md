# ГНЕЗДО - Платформа поддержки родителей

Docker: https://hub.docker.com/r/olegsplaytv/gnezdo

**ГНЕЗДО** - это цифровое пространство для родителей, где они могут получить поддержку, полезную информацию и найти специалистов.

## 📋 Описание проекта

Платформа предоставляет:
- 📚 **Статьи** - проверенная информация о психологии, развитии детей и заботе о себе
- 💬 **Форум** - общение с другими родителями, обмен опытом
- 👨‍️ **Специалисты** - поиск психологов и запись на консультации

## 🎯 Целевая аудитория

- Мамы в декрете (25-35 лет)
- Папы-одиночки
- Молодые родители без опоры на старшее поколение
- Родители подростков

## 🚀 Быстрый старт

### Предварительные требования

- .NET 8.0 SDK
- Docker и Docker Compose (опционально)

### Запуск без Docker

```bash
# Клонировать репозиторий
git clone https://github.com/yourusername/gnezdo.git
cd gnezdo

# Восстановить зависимости
dotnet restore

# Применить миграции БД
dotnet ef database update

# Запустить приложение
dotnet run
